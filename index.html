A weboldal tartalma a LumaR co. tulajdona. Az oldal bármely tartalmának felhasználása jogi következményeket von maga után. Azonban, engedély fejében az oldal tartalma szabadon felhasználható bizonyos keretekig. A szoftverek nem minden esetben tiszták és legálisak. A törvénytelenség elkerülése érdekében minden forrás megtalálható a weboldal láblécében. Megértését köszönjük!

Szoftverek, amiket megtalálsz nálunk!

FONTOS! Csak betekintések! A szoftvereket a 'szoftverek' menüponton éri el és töltheti le.
Visual Studio Code INGYENES
Egy olyan szoftverről beszélhetünk, amelyben jópár nyelven tudunk programkódot írni. Rengeteg Extension (Bővítmény) érhető el, az alkalmazáson belül, amik segítségünkre készültek, hogy minél egyszerűbb legyen programokat készíteni. Jelenleg is a világ egyik legnépszerűbb Code Editor-a (Kódszerkesztője).
PyCharm INGYENES / FIZETŐS
A JetBrains által készített szoftver egy nagyon letisztult python kódszerkesztő. Ez csak egy a több JetBrains által készített és folyamatos frissítésekkel ellátott szoftverek közül. A PyCharm Community Edition (*Közösségi kiadás) Ingyenes, de az eredeti kiadás nem ingyenes.
Visual Studio INGYENES / FIZETŐS
A Visual Studio a Microsoft által fejlesztett integrált fejlesztőkörnyezet (IDE), amely széles körű támogatást nyújt számos programozási nyelvhez és alkalmazás típushoz. A Visual Studio egy erőteljes eszköz, amelyet fejlesztők használhatnak alkalmazásaik tervezéséhez, kódolásához, teszteléséhez és hibakereséséhez.
Github Desktop INGYENES
A Github Desktop a CMD-s megoldást hivatott megkönnyíteni. Letisztult felülettel rendelkezik, jól van optimalizálva és a fejlesztői környezetekkel is remekül kommunikál. Tökéletes alkalmazás azoknak, akik nem szeretnének az alap, parancssoros megoldással meggyürkőzni. Többféleképpen alkalmazhatóak a benne rejlő funkciók.
